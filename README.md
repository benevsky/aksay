# ХБК Аксай — Официальный сайт

Хлебобулочный комбинат «Аксай» — с 1988 года, Алматы.

## Структура проекта

```
aksay-site/
├── index.html          # Главный файл (React 18 + Babel)
├── AksayLogo.svg       # Логотип SVG
├── img/
│   ├── stores/         # Фото фирменных магазинов (8 + заглушка)
│   ├── products/       # Фото продукции (34 шт, 400x400)
│   ├── archive/        # Архивные фото (9 шт, история)
│   ├── production/     # Фото производства (7 шт)
│   ├── certificates/   # Сертификаты (10 шт)
│   └── partners/       # Логотипы партнёров
├── css/                # (зарезервировано)
├── js/                 # (зарезервировано)
└── .gitignore
```

## Запуск

Открыть `index.html` в браузере или через любой локальный сервер:

```bash
npx serve .
# или
python3 -m http.server 8000
```

## Технологии

- React 18 (CDN + Babel in-browser)
- Vanilla CSS (custom design system)
- 4 языка: RU, KK, EN, ZH
- Тёмная тема
- Корзина + форма заказа
